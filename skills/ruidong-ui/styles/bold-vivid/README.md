# bold-vivid

> 强识别度、反平庸——每个界面都应有独特的视觉记忆点。

## 一句话叙事

**让每一个睿动智能体都有自己的"脸"，而不是 AI 批量生产的同款灰蓝界面。**

## 包含子主题

| 子主题 ID | 气质 | 典型场景 |
|---|---|---|
| `cyberpunk` | 赛博朋克，暗底+霓虹双色 | 技术型 Landing Page、AI 工具站 |
| `brutalist` | 野兽派，高对比+粗边框 | 开发者工具、品牌官网 |
| `vaporwave` | 蒸汽波，渐变紫粉+复古感 | 创意工作室、营销活动页 |
| `nordic` | 北欧简约，深灰底+衬线+冷雅 | 文档类产品、咨询类官网、作品集 |

## 适用场景

✅ Landing page · 营销站 · 作品集 · 品牌官网 · 活动页

## 不适用场景

❌ 长时间工作的管理后台（视觉强度高，长时间使用疲劳）  
❌ 需要"克制专业感"的企业工作台（用 `smartsolu-linear` 代替）
